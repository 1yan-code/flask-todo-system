# Flask 待办事项管理系统 ✅
基于 Python Flask + 原生前端（HTML/CSS/JS）实现的全栈待办管理系统，支持「JSON文件存储」和「MySQL数据库存储」双版本，功能完整、代码简洁，适合软件工程入门练手。

## 📋 项目功能
- ✅ 待办事项增/删/改/查
- ✅ 待办完成状态切换（标记完成/取消完成）
- ✅ 数据持久化（JSON/MySQL可选）
- ✅ 前端页面美化，交互友好
- ✅ 完整的前后端分离架构

## 🛠️ 技术栈
### 后端
- Python 3.8+
- Flask 2.0+
- pymysql（MySQL数据库操作）
### 前端
- HTML5 + CSS3 + JavaScript（原生，无框架）
### 数据存储
- 方案1：JSON文件（入门版，0依赖）
- 方案2：MySQL 5.7+/8.0+（工程版，企业标准）
